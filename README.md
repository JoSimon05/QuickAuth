## LOCAL STORAGE
All account secret-keys are stored inside a local database, sorted alphabetically (by name) and reloaded on application startup.

```json
"accounts": [
    {
        "name": "Discord",
        "key": "YOURDISCORDSECRETKEY"
    },
    {
        "name": "Epic Games",
        "key": "YOUREPICGAMESSECRETKEY"
    },
    {
        "name": "Github",
        "key": "YOURGITHUBSECRETKEY"
    }
]
```
